# LuminoPal

LuminoPal is a team project with two parts:

1. ESP32 firmware using ESP-IDF
2. Django web app backend

Repository:
https://github.com/karanrobo/LuminoPal.git

## Project Structure

- esp32_lamp: ESP-IDF firmware project
- server/luminopal_web: Django project

## Prerequisites

### Required Software

1. Git
2. Python 3.10+ (3.11 recommended)
3. ESP-IDF (v5.2.x recommended)
4. VS Code (optional, but recommended)
